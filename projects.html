GSync
One of the more useful, yet extrememly simple things I've made. GSync provdes revision control for rclone remotes that lack it. Google Drive is the most notable example, and is typically what I use this program with. Simply put this program in a crontab, and let it automatically back up your folders with revision control.
SHBM
Simple HeartBeat Monitor is a django server that monitors the uptime of servers using a heartmonitor method. The client's cron uses cURL to make a GET request, resetting a timer on the server side. If that timer reaches 1 minute and 1 second, an email is sent. The extra one second is to avoid high pings causing an alert.
